Customer Churn & Revenue Retention Dashboard
This project showcases an interactive Power BI dashboard built to analyze customer churn, revenue loss, and retention behavior. It combines multiple data sources using SQL and Power Query to deliver actionable business insights.
![Final](https://github.com/user-attachments/assets/d2fd8807-7ca6-47ac-9697-67b8cf7a119e)

________________________________________
**Data Sources**
The dashboard is built using five integrated datasets:
•	customer_master_data – Customer demographics and account details 
•	billing_data – Billing history, revenue, and payment information 
•	customer_support_data – Support tickets and issue resolution metrics 
•	offer_campaign_data – Marketing campaigns and offer performance 
•	usage_data – Customer usage patterns and activity levels 
________________________________________
**Key Features**
•	Churn Overview 
o	Total churned customers: 125 
o	Lost revenue: 81K 
o	Average bill amount: 59.48 
•	Customer Insights 
o	Churn distribution by Prepaid vs Postpaid 
o	Region-wise and segment-wise churn analysis 
•	Campaign Effectiveness 
o	Redemption rates across campaigns 
o	Offer type performance comparison 
•	Trend Monitoring 
o	Monthly active vs inactive customer trends 
o	Revenue fluctuation analysis 
•	Support Analysis 
o	Resolution time by issue category 
o	Identifies service improvement areas 
________________________________________
**Tools & Technologies**
•	Power BI – Dashboard design and visualization 
•	SQL – Data extraction, joins, and transformations 
•	Power Query – Data cleaning and modeling 
________________________________________
**Data Modeling Approach**
•	Combined multiple datasets using common keys (e.g., Customer ID) 
•	Created relationships between fact and dimension tables 
•	Applied transformations such as: 
o	Handling missing values 
o	Aggregating monthly metrics 
o	Creating calculated columns/measures (DAX) 
________________________________________
**Business Impact**
This dashboard helps:
•	Identify high-risk churn customers 
•	Optimize marketing campaigns 
•	Improve customer support efficiency 
•	Reduce revenue loss through better retention strategies
